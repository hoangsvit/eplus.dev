## **Challenge scenario**

You are a Solutions Developer participating in a weekend game jam. Your initial goal is to build a simple **Fantasy Character Creator** to be used in a new fantasy game. As the jam progresses, you'll get new requirements to iteratively add features—like character portraits, backstories, and stats—until you have a complete, interactive character playing card prototype.

**Note:** AI-generated results in Google AI Studio can vary with each run. The goal of this challenge lab is to prototype and explore what's possible, not to build a fully polished application.

### Your challenge

The real test: can you "vibe code" your way through this entire challenge, bringing the app to life using only natural language prompts without writing a single line of code?

Your tasks will be to:

*   Build the initial V1 of the Fantasy Character Creator app.
    
*   Add a feature to generate and regenerate character portraits.
    
*   Refine the app's user interface with a fantasy theme.
    
*   Add a feature to generate character backstories.
    
*   Transform the UI into a player card with stats and add a feature to save characters to a deck.
    
*   Save and download the final application prototype.

After completing the steps in this task, **save** your application. This allows the lab to verify that the required changes have been made.

## Solution of Lab

### Manual

**Task 1:**

```plaintext
Create an interactive web app with a button that generates a random fantasy character when clicked. 
Each character should have a random Name and a Class (like Warrior, Mage, Rogue). 
Display the character's name and class in the app when the button is clicked.
```

**Task 2:**

```plaintext
Add a feature to generate and display a cartoon or video-game-style portrait for the generated character. 
Include two buttons: "Generate Portrait" and "Regenerate Portrait". 
Each portrait should match the character's class.
```

**Task 3:**

```plaintext
Refine the user interface with a fantasy theme.
Give the app a dark, textured background that looks like an ancient alchemist's workbench.
Use a fantasy-style font for the character's name, such as "Uncial Antiqua" or "Cinzel Decorative".
Keep the layout clean and readable.
Add a glowing border or subtle magical particle effect around the main content.
```

**Task 4:**

```plaintext
Add a "Generate Backstory" button. When clicked, create a unique, one-to-two-sentence origin story for the current character and display it.
```

**Task 5:**

```plaintext
Reformat the UI as a bordered player card. Add random stats: Health, Mana, and Strength. Include a "Save to Deck" button that saves the current character to a 'My Deck' list in the app.
```
